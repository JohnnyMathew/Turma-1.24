Exemplos de Código HTML

⮞ Snippet 1: Título e Parágrafo

Este snippet demonstra como usar tags básicas HTML para criar um título e um parágrafo.

        < !DOCTYPE html >
        < html lang="en" >
        < head >
            < meta charset="UTF-8" >
            < title >Minha Página< /title >
        < /head >
        < body >
            < h1 >Título Principal< /h1 >
            < p  >Este é um parágrafo simples.< /p >
        < /body >
        < /html >

⮞ Snippet 2: Lista Não Ordenada

Aqui está um exemplo de como criar uma lista não ordenada utilizando as tags < ul > e < >li >.

        < ul >
            < li >Item 1< /li >
            < li >Item 2< /li >
            < li >Item 3< /li >
        < /ul >

⮞ Snippet 3: Link

Este exemplo mostra como adicionar um link usando a tag < a >.

        < a href="https://www.example.com" >Visitar Exemplo< /a >
